# Build Your Own Arcade Controls FAQ (arcadecontrols.com)

> Reference link to the classic arcadecontrols.com FAQ covering how to design and build arcade control panels (joysticks/buttons), wiring/encoders, layouts, parts sourcing, and practical considerations for building a home arcade cabinet or controller.

## Link
- https://arcadecontrols.com/arcade.htm

## What this is
- A long-running FAQ/reference for building [[Arcade Control Panels]] and related DIY arcade projects.
- Useful for anyone building a [[MAME]] cabinet, a standalone fightstick/arcade controller, or upgrading an existing cabinet.

## Likely topics covered (why it’s useful)
- Selecting parts: joysticks, pushbuttons, microswitches
- Control panel layout and ergonomics (player spacing, button arcs, trackball/spinner placement)
- Wiring basics and ground chains
- Interface options (keyboard encoders, USB game controller boards, etc.)
- Common pitfalls, troubleshooting, and build tips

> [!INFO] When to reference this
> - Choosing a control layout before cutting/drilling a panel
> - Deciding between encoder/interface options for PC-based emulation
> - Learning wiring best practices to reduce noise and simplify maintenance
